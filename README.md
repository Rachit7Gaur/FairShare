# FairShare
FairShare is a lightweight, user-friendly web application that simplifies group expense management.  it helps friends, roommates, and teams track shared costs and settle balances fairly.

## 🚀 Features
- **User Authentication**
  - Local login with username & password
  - Google OAuth 2.0 login
- **Expense Management**
  - Create groups and add members
  - Record expenses and automatically split costs
  - Track balances and settlements
- **UI/UX**
  - Responsive design with Bootstrap
  - Modern frosted glass effects, gradients, and recruiter‑ready polish
- **Team Page**
  - About section showcasing project mission and team members

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap, EJS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** Passport.js (Local & Google OAuth 2.0)
- **Deployment:** Render
